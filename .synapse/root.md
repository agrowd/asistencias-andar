# 🌳 Project Root: Asistencias Andar

## 📌 Descripción
Sistema de gestión de asistencias para la organización "Andar", migrando de un sistema basado en Excel a un CRM web interactivo.

## 🏗️ Arquitectura (Confirmed)
- **Frontend**: Vite + React
- **Backend**: Node.js + Express
- **DB**: SQLite (Local) / Neon PostgreSQL (Production)
- **Aesthetics**: Premium Glassmorphism, Responsive.

## 📂 Estructura de Archivos Principal
- `/` - Raíz del proyecto
- `ASISTENCIA 2026.xlsx` - Fuente de datos original
- `.synapse/` - Memoria persistente de Ariadne Engine
